Napište HTML a CSS kód, který bude odpovídat vizuálu - viz screenshot zadani_v2.png.
Zadaný vizuál odpovídá stránce po načtení (tzn. ihned po načtení stránky by
měla stránka odpovídat vizuálu).
Pár doplňující informací:
1) HTML soubor uložte do složky s názvem "reseni", která se nachází vedle tohoto
textového souboru. Žádné další soubory nikam nepřidávejte a nepřesouvejte.
2) Odsazení levé části s formulářem odpovídá 20 pixelům, šířka levé části pak
250 pixelům, výška pak 500 pixelům.
3) Pravá část (s textem) pak má rozměry 500x500 pixelů.
4) Je žádoucí, aby nám fungovalo automatické číslování u Položka1, Položka2, ...
5) všímejte si detailů (zarovnání textu apod.)
6) Šířka obrázku odpovídá 120 pixelům.
7) Dávejte přednost CSS stylizacím před použitím atributů.
8) Všímejte si detailů (zarovnání textu apod.).
9) Až budete mít práci hotovou, celou složku zadani V2 zazipujte a odevzdejte do
Moodle kurzu PGA.
10) Hodně štěstí!
